Proyecto: Dinosaurio de Chrome 🦖
Descripción

El objetivo del juego es sobrevivir el mayor tiempo posible controlando al dinosaurio de Chrome, evitando los obstáculos que aparecen en el camino.

Instrucciones para jugar
Pulsa la barra espaciadora para saltar.
Evita los cactus y obstáculos que aparecen en el camino.
El juego termina cuando chocas contra un obstáculo.
Intenta conseguir la mayor puntuación posible.
Objetivo

Sobrevivir el mayor tiempo posible y superar tu propio récord.
